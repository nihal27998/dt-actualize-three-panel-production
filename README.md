#  DT Actualize – Hiring Funnel & Assessment System

> A product-driven hiring system designed to process high applicant volume through a structured, data-backed selection funnel.

---

##  Context
DT Actualize was built to handle large-scale hiring with **high selectivity and structured evaluation**.

The goal was to move from an unstructured hiring process to a **scalable, data-driven system** that ensures quality selection.

---

##  Problem
Before DT Actualize:
- Hiring process was unstructured and inconsistent  
- No clear filtering mechanism for large applicant volume  
- High manual effort in screening and evaluation  
- Lack of visibility into candidate progress  

This resulted in:
- Inefficient shortlisting  
- Decision delays  
- Inconsistent hiring quality  

---

##  Solution
Designed and implemented a **multi-stage hiring funnel system** that:
- Filters candidates progressively  
- Tracks performance at each stage  
- Enables structured and objective evaluation  

---

##  Product Thinking Approach
- Identified key stakeholders:
  - Hiring team  
  - Evaluators  
  - Candidates  

- Focused on:
  - Clear stage-wise progression  
  - Measurable evaluation criteria  
  - Reducing manual decision dependency  

- Designed system to answer:
  → *“Who are the top candidates based on structured evaluation?”*

---

## ⚙️ System Design

###  Core Components
- Application intake system  
- Multi-stage evaluation funnel  
- Candidate tracking system  
- Performance-based filtering  

###  Workflow
1. Candidate application intake  
2. Initial screening and filtering  
3. Task-based evaluation stages  
4. Performance tracking across stages  
5. Final selection based on structured criteria  

---

##  Key Features
- Multi-stage hiring funnel  
- Structured candidate evaluation  
- Centralized tracking system  
- Clear visibility into candidate progress  

---

##  Impact
- Processed 1000+ applicants  
- Achieved highly selective hiring (~0.2% selection rate)  
- Reduced manual screening effort  
- Improved consistency in selection decisions  

---

##  Tools Used
- Lovable (no-code platform)  
- Google Sheets (tracking and data structuring)
